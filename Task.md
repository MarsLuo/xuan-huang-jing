一、数值骨架（0→30 min→2 h→离线 8 h）
------------------------------------------------
1. 主线成长公式  
境界突破修为 = 1000 × 1.55^境界（境界 0–9）  
例：  
0→1：1000  
8→9：≈ 31 000  

2. 功法产出曲线（单位：修为/秒）  
| 功法品阶 | 基础修为/s | 30 min 累计 | 2 h 累计 | 8 h 离线累计 |
|----------|------------|-------------|----------|--------------|
| 黄阶下品 | 0.5        | 900         | 3 600    | 14 400       |
| 黄阶上品 | 1.2        | 2 160       | 8 640    | 34 560       |
| 玄阶下品 | 3.0        | 5 400       | 21 600   | 86 400       |

3. 冒险掉落节奏（保底设计）  
玩家每 10 min 可派一次冒险：  
- 70 % 空手而回  
- 20 % 黄阶功法（随机上下品）  
- 9 % 黄阶上品  
- 1 % 玄阶下品  

4. 存储/读取快照  
快照频率：玩家退出、每 60 s、境界升级时各存一次。  
存档内容：  
{“timestamp”:1709999999, “realm”:2, “exp”:5180, “skills”:[“黄阶下品”,“黄阶上品”], “bag”:[“灵石*122”,“玄铁*3”]}

二、项目需求列表（Godot 4.x，竖屏 720×1280）
------------------------------------------------
1. 存储系统（SaveSystem）
- 1.1 全局自动存档：退出/60 s/升级时触发  
- 1.2 读档：启动时若存在 save.dat 则加载，否则新建  
- 1.3 存档结构：JSON，字段见上；提供 save_game()、load_game() 两个静态函数  

2. 修炼系统（Cultivation）
- 2.1 修为每秒自增：sum(所有已学功法修为/s)  
- 2.2 离线补偿：启动时根据上次 timestamp 与当前差值，一次性追加修为  
- 2.3 境界突破按钮：当修为 ≥ 需求值，按钮可用；突破后修为清零，realm+1  
- 2.4 境界显示：顶部 Label “境界 · 炼气期 3 层”  

3. 背包系统（Inventory）
- 3.1 一页 20 格，溢出自动堆叠（功法不可堆，材料可堆）  
- 3.2 双击功法图标→“学习”，若已学则提示“已领悟”  
- 3.3 数据层：数组 inventory[]，元素 {id, count, type}  
- 3.4 UI 层：GridContainer + TextureRect 图标 + Label 数量  

4. 冒险系统（Adventure）
- 4.1 主界面按钮“外出历练”，冷却 10 min（倒计时 Label）  
- 4.2 点击后弹出结果面板：  
  - 成功：获得随机功法/材料，自动入背包  
  - 失败：仅文本提示“此次一无所获”  
- 4.3 掉落表与概率见“数值骨架-3”  
- 4.4 冷却用 Timer 节点，持久化写入存档，读档时恢复剩余时间

三、Godot 节点/脚本速查（伪代码 10 行内）
------------------------------------------------
SaveSystem（AutoLoad）
```gd
static func save_game():
    var data = {
        "ts": Time.get_unix_time_from_system(),
        "realm": Global.realm,
        "exp": Global.exp,
        "skills": Global.skills,
        "bag": Global.bag
    }
    FileAccess.open("user://save.dat", FileAccess.WRITE).store_string(JSON.stringify(data))
```

CultivationPanel（Control）
```gd
func _process(delta):
    Global.exp += Global.exp_per_sec * delta
    exp_label.text = str(snapped(Global.exp,0.1))
```

AdventureButton（TextureButton）
```gd
func _on_pressed():
    if Timer.time_left > 0: return
    Timer.start(600) # 10 min
    var dice = randi() % 100
    if dice < 70: show_result(false)
    else: show_result(true, drop_table[dice])
```
